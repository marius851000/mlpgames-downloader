## MLPGames mirroring tool

Used to run the Scrogne mirror mirror

### Make a mirror:
run ``downloaded.py``. You can use ``--base-url https://mlpgames.mariusdavid.fr/`` to download from Scrogne, which should be faster and less under load.

### Host the mirror:
Run the rust app. You can also simply serve the downloaded folder via your web server, but it won’t have the proper file name when downloaded.
